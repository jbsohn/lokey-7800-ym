This directory contains tools for analyzing the YM2149 sound chip bus activity on the Atari 7800 using a Logic Analyzer.

## `ym2149_analyzer.py`

This script parses a binary capture from `sigrok-cli` and verifies the health of the hardware communication.

### Features
- **Pulse Width Measurement**: Checks if BDIR/BC1 pulses meet the 300ns datasheet requirement.
- **Burst Analysis**: Verifies that "Quad-Tap" or "Double-Tap" software writes are reaching the chip.
- **DMA Shadowing Detection**: Identifies if pulses are being "chopped" or blocked by the Maria graphics chip.
- **Address Selection Stats**: Shows the distribution of Address vs Data writes.

### Usage

1. **Capture Data**:
   Use `sigrok-cli` to capture 8 channels at 24MHz (or higher):
   ```bash
   sigrok-cli -d fx2lafw --config samplerate=24M --samples 24000000 -O binary -o capture.bin
   ```

2. **Analyze**:
   ```bash
   python3 tools/ym2149_analyzer.py capture.bin
   ```

### Channel Mapping
The script assumes the following pin mapping on your Logic Analyzer:
- **D0**: Atari A15
- **D1**: Atari A14
- **D2**: Atari A0
- **D3**: Atari HALT
- **D4**: Atari R/W
- **D5**: Atari PHI2 (CPU Clock)
- **D6**: GAL Pin 17 (BC1 output)
- **D7**: GAL Pin 18 (BDIR output)
